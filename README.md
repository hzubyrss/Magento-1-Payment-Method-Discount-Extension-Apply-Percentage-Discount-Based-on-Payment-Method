# Magento-1-Payment-Method-Discount-Extension-Apply-Percentage-Discount-Based-on-Payment-Method
Magento 1 Payment Method Discount Extension allows store owners to apply automatic percentage discounts when customers select a specific payment method at checkout. Ideal for promoting Zelle, bank transfer, or custom payments. Free and open-source module

This Magento 1 Payment Method Discount Extension allows store owners to automatically apply a percentage-based discount on the cart total when a specific payment method is selected during checkout.

The module helps merchants encourage customers to use preferred payment methods such as Zelle, Bank Transfer, or custom offline payment options by offering an automatic discount incentive.

The extension is lightweight, easy to install, and fully compatible with Magento 1.x stores.

Features

Apply percentage discount on cart total based on payment method

Configure discount percentage from Magento admin panel

Set minimum and maximum order totals

Enable payment method for all countries or selected countries

Display custom instructions for customers during checkout

Configure default order status (Pending / Processing)

Record selected payment method in order view for admin reference

Clean and modular Magento 1 extension architecture

Simple installation without modifying Magento core files

Example

Store owners can offer 3% discount when customers choose Zelle Payment during checkout.
The discount is automatically applied to the cart total and displayed in the order summary.

Compatibility

Compatible with:

Magento 1.7.x

Magento 1.8.x

Magento 1.9.x

Installation

Upload the extension files to your Magento root directory.

Clear Magento cache from System → Cache Management.

Navigate to System → Configuration → Payment Methods.

Configure the payment method and discount settings.
